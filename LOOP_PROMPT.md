## Шаг 4 — Реализуй

Создай все файлы указанные в задаче. Следуй спецификации в CLAUDE.md точно.

Критические правила (нарушение = баг):
- Весь Python I/O — async/await. Никаких синхронных db.query() или requests.get()
- Pydantic v2: `model_config = ConfigDict(from_attributes=True)`, `@field_validator`, не `orm_mode`
- Embedding модель — синглтон, инициализируется один раз при старте приложения, не в каждом запросе/задаче
- TypeScript — строгий, no `any`. Все типы описаны явно
- Gravity UI: в `main.tsx` обязательно `import '@gravity-ui/uikit/styles/fonts.css'` и `import '@gravity-ui/uikit/styles/styles.css'`, обёртка в `<ThemeProvider theme="light">`
- Лимит 50MB: проверять и в Nginx (`client_max_body_size 50m`) и в FastAPI роутере
- Секреты только из `.env` через Settings, никогда хардкодом

## Шаг 5 — Проверь

Выполни проверку указанную в задаче. Если проверка не проходит — исправь, не переходи дальше.

Для Backend задач:
```bash
docker-compose up -d postgres redis qdrant minio 2>/dev/null || true
# запусти сервис и проверь endpoint или импорт
```

Для Frontend задач:
```bash
cd frontend && npm install --silent && npm run build
# убедись что TypeScript компилируется без ошибок
```
